# AstroWatch

# 🚧 NOTE: PROJECT IS IN DEVELOPMENT. INFORMATION MAY BE INACCURATE. 🚧

# ⚠️ THE CURRENT STATE OF THE PROJECT DOES NOT REPRESENT ITS FINAL QUALITY ⚠️

AstroWatch is an experimental platform built to monitor live ISS telemetry, near-earth objects and solar system planet cards.



---

# Features

* ISS speed, altitude, coordinates
* Astronauts currently in orbit now
* Scans NEOs in the last 7 days using NASA API

---


# Cloning The Repository

Open Command Prompt and paste:

```cmd
git clone https://github.com/awkeningdough-ctrl/AstroWatch.git
```

Then enter the project folder:

```cmd
cd AstroWatch
```

---

# Launching The Project

## Option 1 — Open In Vscode

```cmd
code .
```

If that does not work on your system, manually open the it in your desirable coding application.

---

## Option 2 — Run From Terminal

```cmd
cd astrowatch
```

proceed

```cmd
start index.html
```
