# AstroWatch

# 🚧 NOTE: PROJECT IS IN DEVELOPMENT. INFORMATION MAY BE INACCURATE. 🚧

# ⚠️ THE CURRENT STATE OF THE PROJECT DOES NOT REPRESENT ITS FINAL QUALITY ⚠️

AstroWatch is an experimental astronaut health monitoring platform built around Apple Watch biometric data, predictive machine learning, and AI-generated health insights.

The project analyzes health metrics such as stress, cardiovascular condition, and cognitive indicators, then converts those predictions into readable recommendations using AI.

Originally created with astronaut safety and long-duration missions in mind, AstroWatch aims to help detect early warning signs before they become serious.

---

# Features

* Apple Watch health data integration
* AI-generated health recommendations
* Stress analysis
* Cardio monitoring
* Cognitive health indicators
* Machine learning prediction system
* SwiftUI interface
* AWS EC2 backend support

---

# Built With

* Swift / SwiftUI
* AWS EC2
* ARM / Graviton
* HealthKit
* Mistral-7B-Instruct
* Random Forest ML models

# Cloning The Repository

Open Command Prompt and paste:

```cmd
git clone https://github.com/awkeningdough-ctrl/AstroWatch.git
```

Then enter the project folder:

```cmd
cd AstroWatch
```

---

# Launching The Project

## Option 1 — Open In Xcode

```cmd
open AstroWatch.xcodeproj
```

If that does not work on your system, manually open the `.xcodeproj` file inside Xcode.

---

## Option 2 — Run From Terminal

```cmd
swift run
```
